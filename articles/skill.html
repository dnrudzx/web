알고리즘

  • 대학교 과정상의 알고리즘을 능숙하게 활용할 수 있습니다.

  • 프로그래머스 Level3 링크

Python

  • 제가 가장 좋아하는 언어입니다.

  • bs4 : 웹 스크래핑

  • PyQt5 : GUI 실행프로그램 만들기

  • pandas, matplotlib : 데이터 활용

  • 그 외 다양한 오픈라이브러리 활용

Javascript

  • HTML, CSS를 포함한 기본적인 클론코딩이 가능합니다.

  • React나 Vue를 사용하지 않은 순수 JS사용이 가능합니다.

  • 현업에 사용되는 React에 관심을 가지고 공부하고 있습니다.

React

  • 컴포넌트를 만들고 관리할 수 있습니다.

  • 리덕스를 공부하고있습니다.

Django

  • MVC패턴에대해 이해하고있고, 프론트앤드페이지와 연결하여 사용할 수 있습니다.

  • 공부하고있습니다.

Mysql

  • 쿼리를 작성하여 사용할 수 있습니다.

  • 프로젝트에서 다양한 방식으로 사용하는것에 대해 공부하고 있습니다.

Git/Github

  • 다른 개발자와 협업이 가능합니다.

  • 버전관리가 왜 필요하고 어떻게 사용해야하는지 알고있습니다.
